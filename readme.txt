Cataclysm:DDAのmodです。

あったらいいな、と思うようなちょっとした小物を追加します。
Add small gadgets that you think would be nice to have.

ハンディファン
屋外で使いやすいよう握り手が付いた小型扇風機です。うちわを使えばいい？非効率的？いいえ、いま涼しければ何だって良いのです。

乾燥たんぽぽ
腐りやすいタンポポを乾燥させて保存可能にし、いつでもタンポポティーを楽しめるようにします。

e-ink防水ケース
e-inkタブレットPCの大きさにピッタリな防水ケースです、UPS化した時のためにほんの少し余裕があります。
これで水に突き落とされてもデータは無事です。持ち主が生きていれば。

折畳段ボールL/M/S
アイテム状態の各種段ボールを[a]使うと、畳んだ状態に変化させることができるようになります。

オキシマウスピース
有害な気体から肺を守る目的で作られたマウスピースです。
煤煙やガスから身を守りたいけど顔が隠れるのはイヤという方向け。
オキシマウスピースは煤煙程度しか防ぎませんが、強化オキシマウスピースはガスも防ぐことができます。

リストバンドポーチ
装備部位としては左腕に付けるタイプの小さなポーチです。
メスや折り畳みナイフなどの小さな刃物なら鞘として使うこともできます。
容量は少ないものの取り出し動作コストが低いので、緊急時に使いたいアイテムを入れておくのに適しています。
